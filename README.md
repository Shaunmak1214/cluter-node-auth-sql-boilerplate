# node-auth-mysql
____
Node(Express), MySql based app with Authentication Boilerplate.\
It can be used in the start of any project to have JWT authentication.


## How to install

1. clone the repo
2. run ```npm install```
3. create ```.env``` file as given in ```.env.example```
4. run ```npm run dev```
